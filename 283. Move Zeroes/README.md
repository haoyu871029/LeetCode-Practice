## 283. Move Zeroes
在一個整數數列中，將所有的 0 移到最後，並保持其他數字的相對順序，且不使用額外的記憶體。

Example
- Input: [0,1,0,3,12]
- Output: [1,3,12,0,0]
### .cpp
- version 1
    > 對 nums 做一次遍歷，過程中利用一個變數 i 來記錄降落位置，若目前數字非 0，就將該數字與降落位置的數字對調。
- version 2
    > 先遍歷一次 nums 計算 0 的總數，
    > 接著遍歷第二遍，用變數 j 計算目前累積出現幾次 0，每次根據 j 值將當前數字往前移 j 位
    > 最後再將剩下的位置補 0

第一種寫法相較於第二種寫法來說，少一次的遍歷，且最後不用再補 0。

### .py
